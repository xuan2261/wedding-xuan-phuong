# V18.2 — Các bước live còn phải thực hiện

## 1. Cập nhật Google Form

Chạy `updateWeddingRsvpContactInfo()` trong Apps Script để đồng bộ 10h00,
Tư gia nhà trai, địa chỉ 346 Nguyễn Huệ và hạn 28/07/2026.

## 2. Bật prefill tên khách

Chạy `getWeddingRsvpPrefillInfo()` từ file
`tools/get-google-form-prefill-entry.gs`. Sao chép `guestNameEntry` vào:

- `config.js`
- `tools/wedding-data.json`

Sau đó chạy lại `python tests/consistency_check.py`.

## 3. Hoàn thiện file lịch

Chưa có giờ kết thúc thật nên v18.2 không tự đoán. Điền hai giá trị:

- `calendar.ceremonyEndTime`
- `calendar.receptionEndTime`

trong `tools/wedding-data.json`, rồi tạo lại ICS.

## 4. Build deploy sạch

```powershell
python tools/build-dist.py
python tests/verify_dist.py
```

Chỉ upload nội dung `dist/` lên GitHub Pages.
